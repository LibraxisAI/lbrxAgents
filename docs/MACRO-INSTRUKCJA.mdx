## 🏎️ Ekspresowy start

1. **Klonuj repo i zainstaluj zależności**
    ```bash
    git clone ... && cd lbrxAgents
    npm install   # w src lub gdzie leży a2a-protocol
    ```

2. **Otwórz dwa terminale:**

    - **Terminal 1: dashboard**
        ```bash
        cd ratatui-dashboard
        ./scripts/dashboard.sh
        ```

    - **Terminal 2: agent demo**
        ```bash
        node examples/hello-agent.js
        # lub uruchom dowolnego agenta z examples/
        ```

3. **Sprawdź czy dashboard widzi Twojego agenta!**

## 👷‍♂️ Twój agent w 1 minutę!

```js
// hello-agent.js
const a2a = require('a2a-protocol');
const agent = a2a.createAgent({
  name: 'HelloAgent',
  description: 'Odpowiadam na ping!',
  capabilities: ['greeting']
});
setInterval(() => a2a.log('HelloAgent is alive!'), 10000);
agent.onMessage(msg => {
  if (msg.content.text === 'ping') {
    agent.sendMessage(msg.sender_id, { text: 'pong' });
  }
});
```
- agent automatycznie pojawi się w `.a2a/discovery/HelloAgent.json`

## ✍️ Dodaj własnego agenta (JS/Node)

1. **Stwórz plik na podstawie `hello-agent.js`**
   - Zmień `name`, `capabilities` i zachowanie.

2. **Odpalenie = pojawienie się agenta**
   - Agent zapisuje kartę do `.a2a/discovery/`
   - Dashboard to czyta i pokazuje!

## 🛠️ FAQ / Troubleshooting

> **Agent nie pojawia się w dashboardzie?**
- Upewnij się, że agent wrzucił swoją „kartę” do `.a2a/discovery/`
- Pliki muszą być `.json` i nie puste

> **Logi nie aktualizują się?**
- Agent powinien pisać logi do katalogu `logs/`
- Odśwież (klawisz `r`)

> **Błędy Node/Rust/py?**
- Sprawdź czy masz wymagane zależności (`npm install`, Rust toolchain)
- Uruchamiaj agenta osobno, dashboard osobno

> **Nie działa na pierwszym uruchomieniu?**
- Skasuj/make dir-y `.a2a/discovery`, `logs` i spróbuj znowu
